For Students ｜ 中高生のための実践的プログラミング

つくる楽しさから、
本物のスキルへ。

中高生という若い時期から、現場で使われているアプリ開発の技術を習得する。入会後1年で、Python を使ったチャットアプリを自分の手で作れるようになります。

講座内容を見る 生徒の声を読む

Concept

趣味で終わらせない、
社会に活きるレベルまで。

趣味として楽しめる教室は増えました。しかし、スキルとして社会で通用するレベルまで学べる場は、中高生にはほとんどありません。BeEngineer は、IT業界で使われる先端技術をもとにしたカリキュラムで、今後10年以上使える実践的なスキルを届けます。

実践的なITスキル

コードを書く技術、エンジニアリングに必須のツール、ユーザーを考えた設計技術まで。学校教育の知識だけでは届かない領域へ。

論理的思考力

目的を達成する手順を筋道立てて考える訓練を日常的に。エラーの原因を突き止める過程が、文系理系を問わない課題解決能力を育てます。

自己表現力

自分のアイデアを現実化する過程で、思い描くものを表現する力が育ちます。主体性の向上にもつながります。

Curriculum

1年間のカリキュラム

1年目は Python と Web アプリ。ゼミの学習内容は学期ごとに積み上がっていきます。

一学期
  • ◆HTML, CSS の基礎文法
  • ◆Python の基礎文法
二学期
  • ◆Webフレームワーク「Django」の基礎
  • ◆CSS の応用
三学期
  • ◆チャットアプリ開発
  • ◆要件定義

BeEn で学べること

  • ◆HTML、CSS、Python など主要な言語の基礎知識
  • ◆Django を活用した Web アプリケーション開発の技術
  • ◆Web サイトの開発
  • ◆チャットアプリの開発
  • ◆問題を解決するためのアプリの機能の設計

さらに長期休暇には、シューティングゲーム開発、LINEBot 開発、AI 制作などのイベント授業を開講します。

Support

ダブルサポート体制

教材を説明する「講師」と、進捗を確認して回る「サポーター」の2人体制。いつでも質問でき、置いていかれません。

パソコンをお持ちでない方には MacBook Air を貸し出しています。負担なく学習を始められます。
